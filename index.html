개인정보처리방침

DailyMind (이하 "앱")은 사용자의 개인정보를 매우 중요하게 생각하며, 사용자의 개인정보를 보호하기 위해 최선을 다하고 있습니다. 본 개인정보처리방침은 앱이 어떤 정보를 수집하고, 어떻게 사용하는지 설명합니다.

1. 수집하는 개인정보 항목 및 수집 방법

DailyMind는 별도의 서버를 운영하지 않으며, 사용자의 개인정보를 외부 서버로 전송하거나 저장하지 않습니다. 모든 데이터는 사용자의 기기에 로컬로 저장됩니다.

앱이 기능을 수행하기 위해 기기 내에서 접근하거나 저장하는 정보는 다음과 같습니다.

  • 일기 데이터: 사용자가 작성한 텍스트, 날짜, 기분 상태 등.
  • 이미지: 사용자가 일기에 첨부하기 위해 선택한 사진 또는 앱 내 카메라로 직접 촬영한 사진. (기기 갤러리 및 카메라 접근 권한 필요)
  • 앱 잠금 설정 정보: 암호 또는 생체 인증 설정 여부.

2. 개인정보의 이용 목적

앱은 수집한 정보를 다음의 목적으로만 이용합니다.

  • 서비스 제공: 일기 작성, 저장, 조회, 통계 제공 등 앱의 본질적인 기능 제공.
  • 앱 잠금 기능: 사용자의 선택에 따라 앱 접근을 통제하기 위해 사용 (암호 또는 생체 인증).
  • 데이터 백업/복원: 사용자의 요청에 따라 JSON 파일 형태로 데이터를 내보내거나 가져오는 기능 수행.

3. 개인정보의 제3자 제공

앱은 사용자의 개인정보를 제3자에게 제공하지 않습니다. 모든 데이터는 사용자의 기기에만 존재하며, 개발자나 다른 기업이 접근할 수 없습니다.

4. 앱 권한 (Permissions)

앱은 원활한 기능 제공을 위해 다음과 같은 권한을 요청할 수 있습니다.

  • 저장공간/갤러리 접근 (Photo Library / Storage): 일기에 사진을 첨부하거나, 데이터 백업/복원 파일을 저장/읽기 위해 필요합니다.
  • 카메라 (Camera): 일기에 첨부할 사진을 앱 내에서 직접 촬영하기 위해 필요합니다.
  • 생체 인증 (Face ID / Touch ID): 앱 잠금 기능을 사용하기 위해 필요합니다. 기기의 생체 인증 기능을 호출할 뿐, 생체 정보를 직접 수집하거나 저장하지 않습니다.

5. 개인정보의 파기

사용자가 앱을 삭제하거나, 앱 내의 '데이터 초기화' 기능을 사용할 경우, 기기에 저장된 모든 데이터는 즉시 삭제됩니다. 별도의 서버에 저장된 데이터가 없으므로, 앱 삭제 시 복구가 불가능합니다.

6. 개인정보처리방침의 변경

본 개인정보처리방침은 법령 또는 서비스 내용의 변경에 따라 수정될 수 있습니다. 변경 사항이 있을 경우 앱의 업데이트 공지 등을 통해 알려드립니다.
